# Career-Ops CN — AI 求职助手（中国版）

## 工作模式：截图/粘贴 JD → AI 评估 → 匹配分析 → 简历定制

这是一个面向中国国内市场的 AI 求职辅助系统。你可以通过最直接的方式使用它：

### 输入方式（三选一）

1. **截图/照片** — 在任何招聘平台（BOSS直聘、拉勾、猎聘、智联等）看到感兴趣的职位，截图发过来即可。我可以通过图片识别提取 JD 全文进行深度分析。
2. **粘贴文本** — 直接复制 JD 文字发过来，零损耗，最精准。
3. **URL 链接**— 部分平台（如公司官网、海外平台）可能可用，国内大平台通常需登录，不保证能获取到。

### 输出内容

收到 JD 后，我会输出 A-G 完整评估报告：

| 维度 | 内容 |
|---|---|
| **A — 岗位概览** | 画像识别、技术领域、职级、工作模式 |
| **B — 简历匹配分析** | JD 要求 ↔ 简历量化指标逐项对比，含差距弥补策略 |
| **C — 级别判断与策略** | 实际职级匹配度、包装策略、低级别预案 |
| **D — 薪酬竞争力** | 市场薪资调研、该公司薪酬口碑 |
| **E — 定制方案** | 简历/LinkedIn Top 5 修改方案 |
| **F — 面试备考** | STAR 故事包装、防线问答准备 |
| **G — 职位真实性** | 幽灵职位检测、公司经营信号分析 |

### 启用方式

在当前对话中说：
> 使用 modes/ 下的中文模式进行评估

或直接发 JD，我会自动识别并启动评估流程。

## 项目结构

```
career-ops-cn/
├── SKILL.md          # 🤖 OpenClaw 专用入口（IM 渠道）
├── AGENTS.md         # 📝 Claude Code / Codex 等通用入口
├── modes/            # 🧠 AI 评估 prompt 体系（中文）
│   ├── _shared.md    #   共享规则、评分系统、中国市场说明
│   ├── oferta.md     #   A-G 完整评估流程
│   ├── apply.md      #   申请表单助手
│   └── pipeline.md   #   批量处理流程
├── web/              # 🌐 Next.js 前端（本地可视化）
├── scripts/          # 🛠️ 核心工具脚本（按功能分类）
│   ├── cv/           #   简历 PDF/LaTeX 生成
│   ├── tracker/      #   投递追踪管理
│   ├── reports/      #   报告编号与归档
│   ├── diagnosis/    #   诊断与健康检查
│   └── utils/        #   业务工具
├── config/           # ⚙️ 个人配置
├── templates/        # 📄 简历 PDF 模板
├── fonts/            # 🔤 PDF 字体
├── data/             # 📊 追踪数据
└── reports/          # 📋 评估报告存档
```

### 重点说明

- **不依赖 Playwright 抓取**：国内招聘平台反爬激进，不尝试自动化，采用人工截图/粘贴输入。
- **本地优先**：所有数据存储在你的本地文件中，不外传。
- **AI 不代替投递**：评估、定制、生成 PDF 都是辅助，最终投递动作由你手动完成。
